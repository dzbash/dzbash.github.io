5分钟完成 PicGo + Github 图床搭建

创建Github仓库

首先必须要有 一个Github的账号，没有的话 注册 一个就可以了

注册之后来到我们账号的个人主页，点击+号，建库

填写相关信息，进行建库

生成Github Token

首先进入设置界面

进入权限设置

这里选择使用 Personal access tokens并点击 Generate new token 生成一个新的 token

填写相关信息，生成 token

注意这里要保存好token，只出现一次，丢失就只能重新生成

配置PicGo

首先可以在 这里 可以下载PicGo，下载之后解压打开PicGo.exe程序

进入程序之后来到图床设置添加新连接

填写相关信息，最后点击 确定即可，要将其作为默认图床的话，点击设为默认图床

到这里就完成图床搭建了，可以测试上传了，有什么问题可以评论

萌新上路，大佬勿喷

之后我学习了新的图床，简单粗暴，直接利用Typora中的上传图片功能，配合CSDN直接使用，在Typora中利用CSDN图床
