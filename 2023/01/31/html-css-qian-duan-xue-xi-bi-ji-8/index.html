8、元素的层级关系

  • HTML中的元素的层级关系决定了它们在页面上的叠放顺序。元素的层级越高，它们就越靠前，反之则越靠后。

  • HTML文档中的元素有三种定位类型：静态定位（static positioning）、相对定位（relative positioning）和固定定位（fixed positioning）。静态定位的元素的层级是由HTML文档的结构决定的，而相对定位和固定定位的元素的层级可以通过使用z-index属性进行控制。

  • 例如，如果一个元素的z-index属性设置为10，那么它将显示在z-index值为5的元素的前面。因此，可以使用z-index属性来控制页面上元素的叠放顺序。

#myDiv1 {
  position: absolute;
  z-index: 5;
}

#myDiv2 {
  position: absolute;
  z-index: 10;
}

​ 在上面的代码中，myDiv2元素的z-index值比myDiv1元素的z-index值大，因此myDiv2元素将显示在myDiv1元素的前面。

9、更改定位元素的层级

  • 更改定位元素的层级可以通过设置元素的z-index属性来实现。

  • z-index属性可以用来控制元素的层级关系。它是一个整数，数值越大，元素的层级就越高，因此元素就会越靠前显示。

  • 例如，如果一个元素的z-index属性设置为10，那么它将显示在z-index值为5的元素的前面。因此，可以使用z-index属性来控制页面上元素的叠放顺序。

#myDiv1 {
  position: absolute;
  z-index: 5;
}

#myDiv2 {
  position: absolute;
  z-index: 10;
}

​ 在上面的代码中，myDiv2元素的z-index值比myDiv1元素的z-index值大，因此myDiv2元素将显示在myDiv1元素的前面。

1、垂直对齐方式

  • HTML 中有多种方法可以实现垂直对齐，具体取决于您希望如何实现。以下是一些常用的垂直对齐方法：
  1. 行内元素：可以使用 vertical-align 属性在行内元素（如文本）中实现垂直对齐。
  2. 表格单元格：可以使用 vertical-align 属性在表格单元格中实现垂直对齐。
  3. 块级元素：可以使用 display: table-cell 和 vertical-align 属性实现垂直对齐，或者使用 flexbox 布局。
  4. 绝对定位元素：可以使用 top 和 bottom 属性绝对定位元素，并使用 margin 属性实现垂直对齐。
